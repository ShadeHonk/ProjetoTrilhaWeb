# ProjetoTrilhaWeb
Consiste em um projeto fictício, com propósito de aprendizagem onde criamos 
três páginas apenas visuais e para a administração da “loja” fictícia  
criamos um crud para esta adicionando marcas e com as marcas criadas 
adicionando os produtos,para isso foi utilizado MySql como banco de dados, para o  
FrontEnd : HTML/CSS/JavaScript/JQueryUI/Ajax/JQuery  e BackEnd : Java/RestFul/MySql  
Com a arquitetura MVC
